## Project Overview
SitBuddy is a web application that connects parents with babysitters.  
The goal of this platform is to help parents find reliable babysitters in their local area and allow babysitters to showcase their experience and receive job opportunities.

---

## Target Users
- **Parents / Families**
  - Looking for babysitters
  - Want safe and trustworthy childcare

- **Babysitters**
  - Looking for job opportunities
  - Want to present their skills and experience

---

## Problem Statement
Parents often find it difficult to quickly identify and trust suitable babysitters.  
At the same time, babysitters may not have a simple way to connect with families.

---

## Proposed Solution
SitBuddy provides a simple platform where:
- Parents can browse and compare babysitters
- Babysitters can create profiles
- Users can connect through booking requests

---

## Core Features (Draft)
- User registration and login  
- Babysitter profile creation  
- Browse and search babysitters  
- Booking request system  
- Review and rating system  

---

## User Flow

### Parent
Register → Browse → View profile → Send request → Leave review  

### Babysitter
Register → Create profile → Receive request → Accept / Reject
